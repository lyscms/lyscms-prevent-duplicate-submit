# lyscms-prevent-duplicate-submit

#### Description
基于SpringBoot2+实现的一个防止重复提交组件，支持多策略模式：ALL、POST、GET、NONE及兼容注解指定接口拦截或者不拦截，防止用户提交时重复点击导致重复数据。
